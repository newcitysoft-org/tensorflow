input_data.py：用于读取目录下数据配置文件。

mnist.py：通用数字训练算法文件

tensorflowdemo.py，tensorflowdemo1.py，tensorflowdemo2.py，tensorflowdemo3.py ：是tensorflow入门demo

tensorflowdemo5.py：（含线性回归算法，2层卷积神经元，计算交叉熵，ADAM优化器梯度最速下降算法）数字识别训练集模型
